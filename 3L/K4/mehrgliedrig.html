Ausmultiplizieren von Polynomen

Die Regel: "Jedes mit Jedem"

Wenn zwei Klammern miteinander multipliziert werden, musst du jedes Glied der ersten Klammer mit jedem Glied der zweiten Klammer multiplizieren.

Zur Veranschaulichung kannst du dir das Flächenmodell eines Rechtecks vorstellen:

Die allgemeine Formel:
(a + b) · (c + d) = ac + ad + bc + bd

Schritt für Schritt Beispiel

Oft kannst du das Ergebnis am Ende noch zusammenfassen (Gleichartige Glieder addieren!). Achte dabei besonders auf die Vorzeichen.

(x + 2) · (x + 3)

1. Ausmultiplizieren:
= x·x + x·3 + 2·x + 2·3

2. Vereinfachen:
= x2 + 3x + 2x + 6

3. Zusammenfassen:
= x2 + 5x + 6
Vorsicht bei Minuszeichen!
(a - 4) · (a + 2) = a2 + 2a - 4a - 8 = a2 - 2a - 8

Profiklasse: Mehrere Variablen

Wenn verschiedene Variablen in den Klammern vorkommen, entstehen oft Variablenkombinationen wie xy oder ab. Gehe genau nach demselben Muster vor und sortiere die Buchstaben alphabetisch (also xy statt yx), damit du gleichartige Glieder besser erkennst!

Beispiel 1: Zwei Variablen (x und y)
(x + y) · (x + 2y)

1. Ausmultiplizieren:
= x·x + x·2y + y·x + y·2y

2. Alphabetisch ordnen und vereinfachen:
= x2 + 2xy + xy + 2y2

3. Zusammenfassen:
= x2 + 3xy + 2y2
Beispiel 2: Mit Vorfaktoren und Minuszeichen
(2a - 3b) · (a + 4b)

1. Ausmultiplizieren (Achtung auf das Minus!):
= 2a·a + 2a·4b - 3b·a - 3b·4b

2. Vereinfachen:
= 2a2 + 8ab - 3ab - 12b2

3. Zusammenfassen:
= 2a2 + 5ab - 12b2
Beispiel 3: Wenn man nichts zusammenfassen kann
(3x - 2y) · (x - 5z)

1. Ausmultiplizieren:
= 3x·x + 3x·(-5z) - 2y·x - 2y·(-5z)

2. Vereinfachen (Minus mal Minus ergibt Plus!):
= 3x2 - 15xz - 2xy + 10yz

Fertig! Da alle Glieder unterschiedlich sind (x², xz, xy, yz), darf hier nichts weiter zusammengefasst werden.

Die drei Abkürzungen der Mathematik

Warum brauchen wir Binomische Formeln?

Wenn du einen Term wie (a + b)2 siehst, weißt du bereits, dass das eigentlich (a + b) · (a + b) bedeutet. Du könntest das nun mühsam mit der "Jedes-mit-Jedem"-Regel ausmultiplizieren. Das dauert aber lang und ist fehleranfällig.

Die Binomischen Formeln sind Abkürzungen! Wenn du sie auswendig kannst, sparst du dir extrem viel Zeit und Rechenarbeit.

1. Binomische Formel (Die Plus-Formel)

(a + b)2 = a2 + 2ab + b2

Die Herleitung (Warum ist das so?):
(a + b)(a + b) = a·a + a·b + b·a + b·b = a2 + ab + ab + b2 = a2 + 2ab + b2

Beispiel 1:
(x + 3)2
Unser a ist das x, unser b ist die 3.
= x2 + 2·x·3 + 32
= x2 + 6x + 9
Beispiel 2 (Profiklasse mit Zahlen und Variablen):
(2y + 5x)2
= (2y)2 + 2·(2y)·(5x) + (5x)2
= 4y2 + 20xy + 25x2
Häufigster Fehler: (a + b)2 ist NICHT einfach nur a2 + b2! Das Mittelstück (2ab) darf niemals vergessen werden!

2. Binomische Formel (Die Minus-Formel)

(a - b)2 = a2 - 2ab + b2

Sie funktioniert exakt wie die erste Formel, nur dass das Vorzeichen beim Mittelstück ein Minus ist. Das letzte Zeichen bleibt ein Plus (denn Minus mal Minus ergibt Plus!).

Beispiel:
(m - 4)2
= m2 - 2·m·4 + 42
= m2 - 8m + 16

3. Binomische Formel (Die Plus-Minus-Formel)

(a + b)(a - b) = a2 - b2

Das ist die einfachste Formel! Wenn die Klammern exakt gleich aussehen und sich NUR durch das Rechenzeichen in der Mitte unterscheiden, fällt das Mittelstück beim Rechnen komplett weg (+ab und -ab heben sich auf).

Beispiel:
(z + 7)(z - 7)
= z2 - 72
= z2 - 49

Der Binomische Endgegner 🏁

Wende die Formeln richtig an. Pass auf die Mittelstücke auf!

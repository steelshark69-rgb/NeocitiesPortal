2. Die lineare Gleichung

Buch S. 75, Definition
Eine Gleichung der Form a · x + b · y = c nennt man eine lineare Gleichung in den Variablen x und y. Jedes Zahlenpaar (x | y), das die Gleichung erfüllt, heißt Lösung.

Geometrische Interpretation (S. 77)

Jede Lösung entspricht einem Punkt im Koordinatensystem. Die Menge aller Lösungen bildet eine Gerade.

Fall 1: a = 0

b · y = c  →  y = c/b

Waagrechte Gerade
(Parallel zur 1. Achse)
Fall 2: b = 0

a · x = c  →  x = c/a

Senkrechte Gerade
(Parallel zur 2. Achse)
Fall 3: c = 0

a · x + b · y = 0

Ursprungsgerade
(Geht durch den Ursprung)

3. Lineare Gleichungssysteme (LGS)

Buch S. 80, Aufgabe 3.22

Ein LGS besteht aus zwei Gleichungen, die gleichzeitig erfüllt sein müssen.

Hotel-Beispiel (3.22):
20 Zimmer (e + d = 20), 32 Betten (e + 2d = 32).
Grafisch suchen wir den Schnittpunkt der beiden Geraden.

Wie viele Lösungen gibt es?

Klicke auf die Fälle, um die Theorie aus S. 80 zu sehen:

Schnittpunkt: Die Geraden schneiden einander. Die Steigungen sind unterschiedlich.
Parallele Geraden: Die Geraden haben die gleiche Steigung, aber unterschiedliche Achsenabschnitte. Sie treffen sich nie.
Identische Geraden: Beide Gleichungen beschreiben dieselbe Gerade. Jeder Punkt der Geraden ist eine Lösung.

4. Rechnerische Lösungsverfahren

Buch S. 82-84

Wähle ein Verfahren zur Demonstration:

Gleichsetzungsverfahren

Ideal, wenn beide Gleichungen bereits nach derselben Variablen (z.B. y) aufgelöst sind.

(I) y = 2x + 1
(II) y = -x + 4

  1. Gleichsetzen: 2x + 1 = -x + 4
  2. Nach x auflösen: 3x = 3  →  x = 1
  3. x in (I) einsetzen: y = 2(1) + 1  →  y = 3
Lösung: (1 | 3)

Einsetzungsverfahren (Substitutionsmethode)

(I) x - 5y = 17
(II) 2x + 3y = -5

  1. Variable isolieren: Aus (I) folgt x = 17 + 5y
  2. In (II) einsetzen: 2(17 + 5y) + 3y = -5
  3. Klammer lösen: 34 + 10y + 3y = -5
  4. Zusammenfassen: 13y = -39  →  y = -3
  5. y einsetzen: x = 17 + 5(-3) = 2
Lösung: (2 | -3)

Eliminationsverfahren (Additionsmethode)

(I) x - 5y = 17
(II) 2x + 3y = -5

  1. (I) mit -2 multiplizieren: -2x + 10y = -34
  2. Gleichungen addieren: (-2x + 10y) + (2x + 3y) = -34 + (-5)
  3. Variable x fällt weg: 13y = -39  →  y = -3
  4. y einsetzen in (I): x - 5(-3) = 17  →  x = 2
Lösung: (2 | -3)

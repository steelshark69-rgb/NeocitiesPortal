Das Geheimnis der Äquivalenzumformung

Stell dir eine Gleichung wie eine Balkenwaage vor, die perfekt im Gleichgewicht ist. Damit sie nicht umkippt, musst du jede Änderung immer auf beiden Seiten gleichzeitig durchführen. Das nennt man Äquivalenzumformung!

➕ Addition

Legst du links ein Gewicht dazu, musst du rechts genau das gleiche Gewicht dazulegen. ( | + Zahl )

➖ Subtraktion

Nimmst du links ein Gewicht weg, musst du es rechts ebenfalls wegnehmen. ( | - Zahl )

✖️ Multiplikation

Verdreifachst du die Menge auf der linken Seite, musst du sie rechts auch verdreifachen. ( | · Zahl )

➗ Division

Halbierst du die linke Waagschale, musst du auch die rechte Waagschale halbieren. ( | : Zahl )

x + 4 = 10
